# JavaScript Interactive Calculator

## 📌 Project Overview
This is a simple **interactive calculator web application** built using HTML, CSS, and JavaScript as part of Week 08 assignment (JavaScript Basics).  
The project demonstrates DOM manipulation, event handling, and basic arithmetic operations.

---

## 🚀 Features
- Basic arithmetic operations: addition, subtraction, multiplication, division
- Clear screen (C) functionality
- Delete last digit (⌫)
- Support for decimal numbers
- Keyboard support:
  - Numbers (0–9)
  - Operators (+, -, *, /, %)
  - Enter or = to calculate
  - Backspace to delete last character
  - Escape to clear screen
- Responsive and modern UI design
- Hover and click animations for better user experience

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## 🎯 What I Learned
- Working with DOM elements
- Handling user events (click, keyboard events)
- Using functions in JavaScript
- String manipulation and evaluation
- Building interactive UI components
